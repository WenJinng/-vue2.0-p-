# camera

描述：这是个仿醒图的p图软件
功能实现：1.26个Instagram滤镜
         2.可以修改对比度，亮度等参数
         3.可以下载图片（由于htmlcanvas2插件不可以识别filter属性所以下载后的图片会有差异）
## Project setup
```
npm install
```
